## Как использовать

1. Скачайте и установите последнюю версию приложения ([Windows](https://github.com/Antosik/lol-guilds-client/releases/download/v0.0.5/League-Guilds-Client-Setup-0.0.5.exe), [Linux](https://github.com/Antosik/lol-guilds-client/releases/download/v0.0.5/League-Guilds-Client-0.0.5.AppImage), [macOS](https://github.com/Antosik/lol-guilds-client/releases/download/v0.0.5/League-Guilds-Client-0.0.5.dmg))
2. Запустите клиент League of Legends
3. Запустите клиент Гильдий
4. :tada:
